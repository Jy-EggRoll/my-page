# 个人主页项目说明

这是一个使用原生 HTML、CSS 和 JavaScript 构建的个人主页项目。

## 项目结构

- `index.html`: 主页面结构
- `style.css`: 样式表文件
- `main.js`: JavaScript 功能实现
- `images/`: 存放背景图片的文件夹（需要自行创建）

## 功能特点

1. 响应式布局设计
2. 毛玻璃效果卡片
3. 背景图片切换功能
4. 固定背景滚动效果

## 使用说明

1. 创建 `images` 文件夹并添加你的背景图片
2. 在 `main.js` 中更新 `wallpapers` 数组，添加你的图片路径
3. 在 `index.html` 中修改个人信息
4. 根据需要调整 `style.css` 中的样式

## 代码说明

### HTML 结构
- 使用语义化标签组织页面结构
- 主要分为背景容器和内容区域两部分

### CSS 特性
- 使用 CSS Grid 布局
- 实现毛玻璃效果（backdrop-filter）
- 响应式设计适配不同屏幕尺寸

### JavaScript 功能
- 实现背景图片切换
- 添加图片切换过渡效果
- 预加载图片提升用户体验
